EverHeal – Smart Health Assistant

EverHeal is a simple, responsive, and lightweight healthcare web application.
It provides health tips, clean UI sections, and a structure ready for future AI/ML features like health chatbots or symptom detection.
This project reflects strong fundamentals in HTML, CSS, JS and showcases frontend development skills.
